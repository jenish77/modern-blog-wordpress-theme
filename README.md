# Modern Blog Theme

A minimal, responsive, and SEO-friendly WordPress theme designed for modern bloggers. Built with a focus on performance, readability, and ease of customization.

## ğŸ› ï¸ Installation

1.  Download the theme folder (`modern-blog-theme`).
2.  Upload the folder to your `wp-content/themes/` directory or zip it and upload via the WordPress Dashboard (**Appearance > Themes > Add New > Upload Theme**).
3.  Navigate to **Appearance > Themes** and click **Activate** on the **Modern Blog Theme**.

## ğŸ¨ Customization

Access the following options via **Appearance > Customize**:

-   **Colors**: Change the **Primary Theme Color** to match your brand.
-   **Footer Options**: Update the **Footer Copyright Text**.
-   **Site Identity**: Upload your site logo and favicon.
-   **Menus**: Set up your primary navigation menu.
